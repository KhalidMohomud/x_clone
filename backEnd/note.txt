express-async-handler


Use: To handle async errors in Express routes without repeating try/catch blocks.

🔧 Without this:

app.get("/user", async (req, res, next) => {
  try {
    const data = await somethingAsync();
    res.json(data);
  } catch (err) {
    next(err);
  }
});


✅ With express-async-handler:

import asyncHandler from 'express-async-handler';

app.get("/user", asyncHandler(async (req, res) => {
  const data = await somethingAsync();
  res.json(data);
}));


🎯 Cleaner code and centralized error handling.
